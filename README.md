# 📊 HR Analytics Dashboard & Employee Attrition Prediction

## 📌 Project Overview

This project focuses on analyzing employee data to identify the key factors influencing employee attrition. It combines Exploratory Data Analysis (EDA), Feature Engineering, Machine Learning, and an interactive Power BI Dashboard to generate meaningful business insights and support data-driven HR decisions.

---

## 🎯 Objectives

- Analyze employee attrition patterns.
- Identify factors affecting employee turnover.
- Perform data preprocessing and feature engineering.
- Build a machine learning model for attrition prediction.
- Create an interactive HR Analytics Dashboard using Power BI.
- Generate actionable business insights for HR management.

## 📈 Business Insights

- Overall employee attrition rate is approximately **20%**.
- Finance department experiences higher employee attrition.
- Analyst roles record higher employee turnover.
- Employees working overtime tend to leave more frequently.
- Middle-aged employees contribute significantly to attrition.
- Employee tenure influences attrition trends.
- Dashboard supports data-driven HR decision-making.
